迅优移动路由优化工具集 (XunYou MZ803)
=========================================

针对迅优 (XunYou) MZ803 系列 4G 移动路由器的优化工具。

为啥写这堆东西?
  花 30 块钱买了个迅优移动路由器，插电信卡上网。
  但这破路由器 WiFi 信号差、信道不会自动优化、切 APN 还得进后台点半天。
  所以写了这套工具，让 macOS 命令行直接管它:
  - 扫周边信道，自动切到最不堵的那个
  - 一键切换 APN (联通/电信/移动随便换)
  - 测速、测国际连通性
  反正路由器的 API 都暴露了，不折腾白不折腾。

依赖:
  - Python 3
  - speedtest (Ookla CLI) - 测速用，已包含在目录中
  - macOS airport 命令 (WiFi 扫描)

文件说明:

  chan        WiFi 信道分析 + 自动切换
              ./chan             扫描分析 + 交互式切换信道
              ./chan --auto      自动切换到最佳信道（一次）
              ./chan --watch     自动切换 + 持续监控（默认每 60 秒）
              ./chan --watch 60  自定义监控间隔（秒）

              切换算法:
                1. 当前信道干扰分 < 100  → 不切（够好就别折腾）
                2. 已有最佳信道           → 不切
                3. 有流量 (rx/tx >500B/s) → 不切（有人在用）
                4. 空闲                   → 切到最佳信道

  apn         APN 管理工具
              ./apn              交互式 APN 管理（查当前、切换）
              ./apn --test        遍历所有 APN 测速 + 排序
              ./apn --open        国际连通性测试（25 个站点并行检测）

  openness.py  国际连通性测试独立脚本（apn --open 调用它）

路由器信息:
  品牌: 迅优 (XunYou)
  型号: MZ803 系列
  管理地址: http://192.168.100.1
